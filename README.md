# Password Strength Checker

A simple **Java program** that checks the strength of a password and gives suggestions to improve it.

## Features

- Checks for:
  - Minimum length (≥8)
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Gives a final strength: Weak / Medium / Strong
- Provides suggestions to make passwords stronger
